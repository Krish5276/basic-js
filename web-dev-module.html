🌐 Web Development Module

A comprehensive 7-days journey through web development fundamentals — from HTML basics to deploying a complete portfolio website. This module covers front-end technologies, responsive design, JavaScript interactivity, and modern web practices.

📅 7 Days ✅ Completed 💻 Front-End Focus
01

Basic HTML Tags

Introduction to HTML Concepts and Understanding Basic Structure of Webpages

I learned how to embed images, videos, and audio files into a webpage.
I understood how to create internal and external hyperlinks for navigation.
Tables were introduced to organize data in rows and columns effectively.
I practiced using table headers, rows, and cells to display structured information.
A webpage was designed incorporating media elements and a table to demonstrate these concepts.

Adding images, videos, audio, hyperlinks, tables, headers, rows, and cells.
Structuring data and embedding multimedia for interactive webpages.

Setting up the development environment

02

HTML Structure & Semantic Elements

Deep dive into HTML5 tags, forms, tables, and semantic markup

Day 2 focused on mastering HTML5 semantic elements. We learned about structural tags like header, nav, main, section, article, aside, and footer. The session included creating forms with various input types, building tables for data display, and understanding the importance of semantic HTML for accessibility and SEO.

Key topics: Semantic HTML5 elements, Forms and input types, Tables and lists, Multimedia elements (images, audio, video), Links and navigation, Accessibility basics with ARIA attributes.

Learning HTML5 semantic elements

03

CSS Fundamentals & Styling

Colors, typography, box model, and basic layouts

Day 3 introduced CSS — the language that brings visual life to web pages. We covered selectors, properties, the box model (margin, border, padding, content), and how CSS cascading and specificity work. We styled our HTML pages with colors, fonts, backgrounds, and borders.

Key topics: CSS syntax and selectors, Box model deep dive, Colors and gradients, Typography and Google Fonts, Backgrounds and borders, CSS units (px, em, rem, %, vh, vw).

Exploring CSS styling techniques

04

Responsive Design & Flexbox

Media queries, Flexbox layout, and mobile-first design

Day 4 was all about making websites look great on every device. We learned Flexbox for creating flexible layouts, media queries for responsive breakpoints, and the mobile-first design approach. We rebuilt our previous pages to be fully responsive across desktop, tablet, and mobile screens.

Key topics: Flexbox container and item properties, Media queries and breakpoints, Mobile-first vs Desktop-first approach, Responsive images and typography, Viewport meta tag, CSS Grid introduction.

Responsive design across multiple devices

05

JavaScript Basics

Variables, data types, functions, and control flow

Day 5 marked the beginning of our JavaScript journey. We learned about variables (let, const, var), data types, operators, conditional statements, and loops. We wrote our first interactive scripts that responded to user actions and manipulated data dynamically.

Key topics: Variables and constants, Data types (string, number, boolean, array, object), Operators and expressions, If/else and switch statements, For, while, and do-while loops, Functions and scope.

Writing first JavaScript programs

06

DOM Manipulation & Events

Selecting elements, modifying content, and handling user events

Day 6 focused on the Document Object Model (DOM) — the bridge between HTML and JavaScript. We learned to select elements using getElementById, querySelector, and querySelectorAll. We practiced modifying content, styles, and attributes dynamically, and handled events like click, hover, keypress, and form submission.

Key topics: DOM tree structure, Selecting and traversing elements, Modifying innerHTML and textContent, Changing styles and classes, Event listeners and handlers, Event bubbling and delegation.

DOM manipulation and debugging

07

Forms & Validation

Building interactive forms with client-side validation

Day 7 was dedicated to creating robust, user-friendly forms. We built contact forms, registration forms, and login interfaces with comprehensive client-side validation using JavaScript. We learned regex patterns for email and phone validation, real-time feedback, and form submission handling.

Key topics: Form elements and attributes, Client-side validation techniques, Regular expressions for validation, Real-time input feedback, Form submission and preventDefault, Custom error messages and styling.

Building forms with validation

08

APIs & Fetch

Working with REST APIs, JSON data, and asynchronous JavaScript

Day 8 introduced us to the world of APIs and asynchronous programming. We learned about REST APIs, JSON data format, and how to use the Fetch API to retrieve data from external services. We built a weather app and a quote generator that fetched real-time data from public APIs.

Key topics: What are APIs, JSON format and parsing, Fetch API and promises, Async/await syntax, Error handling with try/catch, Displaying API data dynamically.

Working with APIs and JSON data

09

Portfolio Project Development

Building a complete personal portfolio from scratch

Day 9 was the culmination of everything we learned. We started building our personal portfolio website from scratch, combining HTML structure, CSS styling, responsive design, and JavaScript interactivity. We implemented sections for hero, about, skills, projects, and contact — applying all concepts learned throughout the module.

Key topics: Project planning and wireframing, Component-based thinking, CSS animations and transitions, Dark/light mode implementation, Smooth scrolling and navigation, Image optimization and lazy loading.

Assembling the complete portfolio

10

Deployment & Final Review

Testing, optimization, and deploying the website live

The final day focused on polishing, testing, and deploying our portfolio websites. We learned about GitHub Pages for free hosting, domain configuration, performance optimization, and SEO best practices. We conducted cross-browser testing, fixed responsive issues, and celebrated our completed projects.

Key topics: Git and GitHub basics, GitHub Pages deployment, Performance optimization (minification, compression), SEO meta tags and Open Graph, Cross-browser testing, Final code review and documentation.

Deploying the website to the world
