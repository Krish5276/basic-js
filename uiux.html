🎨 UI / UX Design Module

A comprehensive 5-day journey through UI and UX design fundamentals.
This module covers visual interface design, brand-based layouts, modern design trends
like Glass Morphism, and user experience principles, ending with a final UI/UX project.

📅 5 Days ✅ Completed 🎨 UI / UX Design
01

Introduction to UI

Understanding the Basics of User Interface Design

On the first day, I learned the fundamentals of User Interface (UI) design,
which focuses on the visual elements that users interact with in digital products.
I explored components such as buttons, icons, typography, and layout structure.
Basic design principles like alignment, spacing, contrast, and consistency
were introduced to create clean and user-friendly interfaces.

Key Concepts:
User Interface basics, layout structure, design consistency,
visual hierarchy, UI components.

Basic UI concepts demonstrating interface elements, layout structure, and visual design.

02

Red Bull Page Design

Designing a Brand-Based Webpage Interface

On the second day, I created a Red Bull themed webpage design
to practice UI layout and branding.
The design focused on bold colors, strong typography,
and a structured layout that reflects the Red Bull brand identity.
I organized the page using sections like header, hero content,
and footer while maintaining visual balance.

Key Concepts:
Brand-based design, color schemes, layout structure,
visual hierarchy, brand identity.

Red Bull themed webpage layout showing brand colors and typography design.

03

Glass Morphism

Learning Modern UI Design Trends

On the third day, I learned about Glass Morphism,
a modern UI design technique used in many contemporary interfaces.
This style creates a frosted glass effect
using transparency, blur, and subtle shadows.
I practiced designing interface cards and panels
to make layouts look modern and visually layered.

Key Concepts:
Glass Morphism effect, transparency, blur effects,
modern UI styling, layered interface design.

Modern UI cards demonstrating glass morphism effect with transparency and blur.

04

Introduction to UX

Understanding User Experience Design

On the fourth day, I was introduced to User Experience (UX) design,
which focuses on improving how users interact with digital products.
I learned about usability, accessibility,
and user-centered design principles.
Understanding user needs and designing simple navigation
helps create smooth and efficient user experiences.

Key Concepts:
User experience fundamentals, usability, accessibility,
user journey, user-centered design.

UX design concepts illustrating user interaction flow and usability principles.

05

Final UI/UX Project

Applying UI and UX Concepts to a Complete Design

On the final day, I worked on a UI/UX mini project
combining all the concepts learned throughout the module.
The project involved designing an interface layout,
applying UI principles, and ensuring a smooth user experience.
This helped me understand how UI visuals
and UX functionality work together in real digital products.

Key Concepts:
UI and UX integration, layout design,
user interaction, design workflow,
practical implementation.

Final UI/UX project showcasing interface layout, design elements, and usability.
