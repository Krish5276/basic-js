🌐 Web Development Module

A comprehensive 7-days journey through web development fundamentals — from HTML basics to deploying a complete portfolio website. This module covers front-end technologies, responsive design, JavaScript interactivity, and modern web practices.

📅 7 Days ✅ Completed 💻 Front-End Focus
01

Basic HTML Tags

Introduction to HTML Concepts and Understanding Basic Structure of Webpages

On the first day, I learned the basic structure of an HTML document.
I explored common tags such as paragraphs, headings, bold, and italic text to format content.
I also learned how to use line breaks and add comments for clarity.
The <marquee> tag was used to create scrolling text effects.
A small webpage was created to apply all these basic HTML concepts practically.

Key Concepts: HTML structure, paragraphs, headings, bold, italic, line breaks, comments.
Marquee tag, semantic organization, and understanding webpage basics.

>

simple HTML page showing headings, paragraphs, bold, italic, and a marquee text.

02

Media Elements and Tables

Learning How to Add Media Elements and Organize Data Using Tables

I learned how to embed images, videos, and audio files into a webpage.
I understood how to create internal and external hyperlinks for navigation.
Tables were introduced to organize data in rows and columns effectively.
I practiced using table headers, rows, and cells to display structured information.
A webpage was designed incorporating media elements and a table to demonstrate these concepts.

Key Concepts: Adding images, videos, audio, hyperlinks, tables, headers, rows, and cells.
Structuring data and embedding multimedia for interactive webpages.

webpage with images, video/audio embedded, and a table displaying data.

03

Forms using html

Understanding Form Creation and Collecting User Input Through Webpages

This day focused on creating forms to gather user input.
I learned different types of input fields such as text, email, radio buttons, and checkboxes.
Labels were added to ensure accessibility and proper form identification.
Submit and reset buttons were used to handle form actions.
A complete contact form was built to implement all these concepts practically.

Key Concepts: Form creation using different input types and labels.
Submit/reset buttons and structuring forms for usability.

contact form with text inputs, radio buttons, checkboxes, and submit/reset buttons.

04

Layout and CSS Selectors

Designing Webpage Layout and Applying Styles Using Different CSS Selectors

I learned how to structure webpages using semantic elements like header, navigation, main content, and footer.
CSS selectors such as element, class, and ID were used to style different parts of the webpage.
I explored properties like color, background, font, and text alignment.
Styling enhanced the appearance and readability of the webpage.
A fully styled webpage layout was created to practice CSS selectors and structured design.

Key Concepts: Semantic layout elements and CSS selectors for styling.
Applying colors, fonts, and backgrounds for improved design.

webpage with styled header, navigation menu, and footer using CSS selectors.

05

Box Model and Flexbox

Understanding Box Model Concepts and Using Flexbox for Layout Design

The box model concept was studied to understand how elements are displayed with margin, border, padding, and content.
I learned how spacing affects the layout of elements on a webpage.
Flexbox was introduced to align items horizontally, vertically, and distribute space efficiently.
Several boxes were created to experiment with alignment, spacing, and responsiveness.
A simple webpage was built to practice both box model properties and Flexbox layout techniques.

Key Concepts: Box model: margin, border, padding, content, element spacing.
Flexbox for alignment, spacing, and responsive layout design.

showing multiple boxes aligned using Flexbox and demonstrating the box model properties.

06

JavaScript Basics

Introduction to JavaScript Programming and Performing Basic Logical Operations

I was introduced to JavaScript as a programming language for web interactivity.
Variables and data types were learned to store different values.
Basic operators and arithmetic operations were applied to perform calculations.
Functions and simple statements like alert() and console.log() were used to display outputs.
Several small programs, such as adding numbers and checking even/odd numbers, were implemented for practice.

Key Concepts: JavaScript variables, data types, operators, alert(), console.log().
Performing basic arithmetic and logical operations using scripts.

browser console showing JavaScript outputs for simple programs.

07

To-Do List Project

Building a Complete Mini Project Using HTML CSS and JavaScript Skills

The final day focused on building a functional project to integrate all concepts learned.
I created a To-Do List application using HTML for structure, CSS for styling, and JavaScript for functionality.
Users can add new tasks, delete tasks, and mark tasks as completed.
This project demonstrated practical application of all previous lessons.
It helped strengthen understanding of web development and combining multiple technologies in a single project.

Key Concepts: Integrating HTML, CSS, and JavaScript for interactive projects.
Event handling for adding, deleting, and completing tasks.

To-Do List application showing tasks being added, marked, and deleted.
